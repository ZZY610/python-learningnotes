## pytorch能做什么
### GPU加速

```python
import torch
import time

a = torch.randn(10000,1000)
b = torch.randn(1000,10000)

t0 = time.time()
c = torch.matmul(a,b)
t1 = time.time()
print(a.device,t1 - t0,c.norm(2))

device= torch.device("cuda")
a= a.to(device)
b= b.to(device)

t0 = time.time()
c = torch.matmul(a,b)
t2 = time.time()
print(a.device,t2 - t0,c.norm(2))

t0 = time.time()
c = torch.matmul(a,b)
t2 = time.time()
print(a.device,t2 - t0,c.norm(2))
```
